## i18n Configuration

Environment keys:
- `I18N_ENABLED=true|false`
- `I18N_DEFAULT_LANG=en`
- `I18N_FALLBACK_LANG=en`

Resources location: `resources/i18n`.

Packages:
- `@forgeon/i18n`
- `@forgeon/i18n-contracts`
- `@forgeon/i18n-web`

You can apply i18n later with:
`npx create-forgeon@latest add i18n --project .`
